PyTutor AI 🐍🧠
An interactive, AI-powered Python learning platform designed to help aspiring developers master Python through hands-on practice, automated feedback, and structured assessments. PyTutor AI bridges the gap between passive reading and active coding by providing a seamless, in-browser development environment guided by artificial intelligence.

🚀 Features
Integrated Code Editor: Write, execute, and test Python code directly within the browser—no local environment setup required.

Intelligent AI Assistant: Get real-time hints, code explanations, and debugging support from a built-in AI tutor that guides you without just giving away the answers.

Comprehensive Quiz System: Test your understanding of Python concepts with dynamic quizzes that reinforce learning and track knowledge retention.

Secure User Authentication: Personalized dashboards and progress tracking so learners can pick up exactly where they left off.

Responsive Design: A clean, accessible UI built for optimal learning across both desktop and mobile devices.

🛠️ Tech Stack
Framework: Next.js & React.js

Styling: Tailwind CSS

State Management: Redux

Deployment: Vercel

AI Integration: Gemini API

⚙️ Getting Started
Follow these instructions to set up the project locally for development and testing.

Prerequisites
Make sure you have Node.js and npm (or yarn/pnpm) installed on your machine.

Installation
Clone the repository:

Bash
git clone https://github.com/smith-michael26/pytutor-ai.git
cd pytutor-ai
Install dependencies:

Bash
npm install

# or yarn install

Set up environment variables:
Create a .env.local file in the root directory and add your required API keys and database URIs:

Code snippet
NEXT_PUBLIC_API_URL=your_api_url_here
AI_API_KEY=your_ai_provider_key_here
DATABASE_URL=your_database_url_here

Run the development server:
Bash
npm run dev

# or yarn dev

Open the application:
Navigate to http://localhost:3000 in your browser to see the app in action.
